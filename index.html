Formula Compendium

Algebra
  • Quadratic Formula: x = [-b ± √(b²-4ac)] / (2a)
  • Vieta: r₁+r₂ = -b/a, r₁r₂ = c/a
  • Discriminant: D = b² - 4ac
  • Factor: a²-b²=(a-b)(a+b)
  • Binomial: (a+b)^n = Σ C(n,k) a^{n-k} b^k
Geometry
  • Circle area: A = πr²
  • Triangle area: A = ½ b h
  • Heron: A = √[s(s-a)(s-b)(s-c)]
Trigonometry
  • SOHCAHTOA: sin=opp/hyp, cos=adj/hyp, tan=opp/adj
  • Law of Sines: sinA/a = sinB/b = sinC/c
  • Law of Cosines: c²=a²+b²-2ab cos C
Calculus
  • Power Rule: d/dx xⁿ = n x^{n-1}
  • Product Rule: d(uv)=u'v+uv'
  • Chain Rule: d/dx f(g(x)) = f'(g(x)) g'(x)
  • Integral: ∫ xⁿ dx = x^{n+1}/(n+1) + C
Linear Algebra
  • Det 2×2: det([[a,b],[c,d]]) = ad - bc
  • Eigen: det(A - λI) = 0
Statistics
  • Mean: μ = Σxᵢ / n
  • Variance: σ² = Σ(xᵢ - μ)² / n
  • Combinations: C(n,k) = n! / (k!(n-k)!)
